# 📞 Phonebook Console Application (C++)

A simple **C++ DSA-based Phonebook system** built using Object-Oriented Programming and File Handling. This project allows users to manage contacts through a console interface.

---
## Author

Abdullah Imran

## 🚀 Features

-  Add new contact
-  Search contact by name
-  Delete contact
-  Save contacts to file
-  Load contacts from file (if implemented)
-  Simple menu-driven interface

---

##  Tech Stack

- Language: C++
- Concepts Used:
  - Object-Oriented Programming (OOP)
  - File Handling
  - STL (map)
  - Basic Data Structures
